# Vue-thinkphp
采用了Vue2+thinkphp实现了物流后台管理系统

## 项目介绍
本项目借鉴了https://gitee.com/GuoJingXun/deer-wms/widgets/widget_3.svg 项目，通过用Vue2+thinkphp改写该项目，更快更高效的数据展示，同时实现了代码的复用。
